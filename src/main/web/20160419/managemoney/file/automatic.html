添益宝自动转入服务协议

添益宝自动转入服务协议（以下简称“本协议”）是天翼电子商务有限公司（以下简称“本公司”或“翼支付”）与翼支付用户(以下简称“您”或“投资者”)就添益宝自动转入服务相关事项所订立的有效合约。 投资者通过网络页面点击确认或以其他方式选择接受本协议，即表示投资者与本公司已达成协议并同意接受本协议的全部约定内容以及与本协议有关的各项规则及本公司网站所包含的其他与本协议或本协议项下各项规则的条款和条件有关的各项规定。 

在接受本协议之前，请您仔细阅读本协议的全部内容（ 特别是以粗体下划线标注的内容）。如果您不同意本协议的任意内容，或者无法准确理解本公司对条款的解释，请不要进行后续操作。 

 

第一条 释义 

1、 添益宝自动转入服务：是指在投资者开通自动转入功能后，翼支付根据翼支付合作的金融机构（包括但不限于保险公司、银行、基金公司、证券公司等，以下简称“翼支付合作金融机构”）的指令，从投资者翼支付账户余额中扣取超出保底金额的款项，由翼支付合作金融机构在翼支付的网上直销自助前台系统将扣取款项自动为投资者完成理财产品的申购的服务。

2、 保底金额，是指您在开通添益宝自动转入服务时设置的阀值。当您的翼支付账户余额超过阀值时，超出部分款项将自动用于申购相应的理财产品。如您的翼支付账户余额低于阀值时，则不作前述自动转入。

3、余额理财申购是定期不定额申购业务的一种，在本协议中是指翼支付合作金融机构直销系统根据投资者设定的翼支付账户保底金额，定期向投资者翼支付账户触发扣款指令，为投资者完成不同金额的添益宝基金申购申请的一种业务。扣款周期为每个自然日（D日）一次。 

4、D日是指自然日。 

5、开放日是指基金公司为投资者办理基金申购、赎回等业务的工作日。 

6、T 日是指符合翼支付合作金融机构规定或其他法律文件规定的投资者申购申请日。 

7、T ＋ n 日是指自 T 日起第 n 个工作日（不包含 T 日）。 

 

第二条 业务规则 

1、余额理财申购交易流程为，您通过本公司网上交易直销自助式前台签订余额理财申购协议，设定翼支付账户保底金额。您开通本业务后，由翼支付合作金融机构系统根据您设定条件于次一自然日开始计算出每期申购金额，定期向翼支付发出扣款指令。 

2、单笔/日累计/月累计最高限额以本公司网上交易直销自助式前台的提示为准。 超过单笔日累计\月累计最高额度的，翼支付合作金融机构直销系统将不发起余额理财申购申请。 

3、翼支付合作金融机构直销系统将在每个D日的凌晨发起一次扣款指令。您签订余额理财申购协议后，需保证翼支付账户在扣款日当天的账户余额高于其设定的保底金额且账户状态正常。当您翼支付账户余额不高于其设定的保底金额、账户状态不正常或网络故障、系统故障不可抗力导致D日无法扣款成功的情况下，该笔余额理财申购交易将不予顺延。 

4、您签订本协议后，若扣款日为非开放日时，则申购申请将顺延至下一申购申请日。申购份额将在 T+1日确认成功后直接计入您的基金账户。 

5、您同意签订本协议并提交后，可终止协议或修改翼支付账户保底金额。余额理财申购协议状态为正常时方可终止协议或修改翼支付账户保底金额，若您是在当日（D日）扣款时间前终止协议或修改翼支付账户保底金额的，当日（D日）不执行自动转入或按照修改后的保底金额执行自动转入；若您是在当日（D日）扣款时间之后终止协议或修改翼支付账户保底金额的，自下一个自然日起不执行自动转入或按照修改后的保底金额执行自动转入。 

6、通过余额理财申购业务成功申购的基金份额的赎回、快速赎回、分红和其他业务遵照《证券投资基金投资人权益须知》相关规定。 

7、翼支付有权暂停添益宝基金的申购业务，在公告的暂停申购期限内将暂停对该基金的余额理财申购交易。 

8、您可通过本公司网上交易直销自助式前台查询余额理财申购协议，以及余额理财申购的交易申请情况。

 

第三条 特别约定

1、您开通添益宝服务，且经翼支付验证符合开通添益宝自动转入服务条件的投资者，可以在设置翼支付账户余额的保底金额后开通添益宝自动转入服务。

2、您授权在您开通添益宝服务的同时，将添益宝增加为翼支付为您提供的各类代扣服务的扣款资金渠道之一，可使用添益宝作为代扣资金渠道的具体代扣服务以翼支付实际向您提供的为准。

您理解并同意，翼支付有权根据实际情况在提供具体代扣服务时决定添益宝在扣款资金渠道中的顺位，具体以页面提示或翼支付网站的相关规则、解释为准。

您理解并同意，当您发送代扣指令（含单笔代扣指令、预授权方式等），且该代扣指令对应的扣款渠道为添益宝时（简称“添益宝代扣”），即会触发添益宝货币基金份额支付的申请。您理解并同意自您发送“添益宝代扣”指令之日起，您将不再承担享有对应的添益宝货币基金份额的损益。

3、您理解并同意，在您开通添益宝自动转入服务后，翼支付在您翼支付账户余额大于您设定的保底金额时，及时向翼支付合作金融机构反馈。为了保障您的资金安全，若您的添益宝中的金额已经达到添益宝上限额度，翼支付将不向翼支付合作金融机构反馈您翼支付账户余额的信息。

4、您授权翼支付根据翼支付合作金融机构的指令从您的翼支付账户中进行扣款，并由翼支付合作金融机构在翼支付的网上直销自助前台系统将相应款项转入您的添益宝。

5、您设置保底金额时，应预测并预留足够额度保障您后续通过翼支付账户余额完成相关交易。如您设置的保底金额过低，您开通添益宝自动转入功能后，可能导致您翼支付账户的余额不足以支付您的其他支付需求，因此给您造成的损失（包括但不限于其他支付需求无法完成，或在代扣服务中因翼支付账户余额不足而直接从您的添益宝中扣取相应款项，可能对您的添益宝收益造成部分损失等）由您自行承担。

6、您理解并同意，对于添益宝自动转入功能的开通、修改、关闭等操作，均需要遵守合作金融机构系统在翼支付的网上直销自助前台系统的相关提示和规则。

7、您理解并同意，翼支付仅向您提供资金划转等服务，您通过“添益宝自动转入”功能转入添益宝的资金的收益计算，以您与翼支付合作金融机构签订的相关协议约定为准，翼支付不参与您与翼支付合作金融机构之间的任何纠纷。

8、您理解并同意，翼支付有权视情形（包括但不限于添益宝对应的理财产品停售、发生较大投资风险等）单方中止/终止向您提供本协议约定的添益宝自动转入服务。翼支付开展前述行动时将以本协议约定的方式通知，而无需另行单独通知您，也不就该等行动给您造成的损失（包括但不限于丧失投资机会而未能取得投资收益）承担责任。

 

 

第四条 风险

因发生不可抗力事件导致余额理财申购业务无法正常进行的，本公司应及时通知其他合同当事人及相关方并积极采取措施予以解决，但不承担赔偿责任。不可抗力包括但不限于战争、火灾、地震、水灾、罢工、网络故障、系统故障等不能合理预见、不可避免和不能克服的自然灾害、社会异常事件及技术风险等。

1、 关于本协议条款和其他协议、告示或其他有关您使用添益宝自动转入服务的通知，本公司将以电子形式或纸张形式通知您，包括但不限于依据您向本公司提供的电子邮件地址发送电子邮件的方式、依据您提供的联系地址寄送挂号信的方式、于本网站或合作伙伴网站上公布、或发送手机短信和电话通知等方式。
